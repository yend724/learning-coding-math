# Creative Codingの積み木

数学や物理をはじめとしたCreative Codingの技法を学ぶブログ。

URL:https://tsumic.yend.dev/

## Install

```shell
$ cd tsumic
$ yarn install
```

## Run locally

Start dev server.

```shell
$ yarn dev
```

## Build

Builds the project for production to the dist folder.

```shell
$ yarn build
```

## Preview

Locally preview production build.

```shell
$ yarn start
```
